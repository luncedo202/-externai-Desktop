# New Folder Creation Feature

## Overview
You can now create new folders in the Explorer sidebar!

## How to Create a New Folder

### Method 1: Using the Folder Header Button
1. Open a workspace folder in the app
2. Look at the Explorer sidebar
3. You'll see a folder name at the top with a **âž•** button next to it
4. Click the **âž•** button
5. An input field will appear
6. Type the folder name and press **Enter**
7. Press **Esc** to cancel

### Method 2: Right-Click on Any Folder (Context Menu)
1. Open a workspace folder
2. Right-click on any folder in the tree
3. An input field will appear to create a subfolder inside that folder
4. Type the folder name and press **Enter**
5. Press **Esc** or click away to cancel
