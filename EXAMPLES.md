### Example 2: Build a React Todo App

1. **Create project folder:**
   ```bash
   # In terminal (Cmd/Ctrl+`)
   mkdir react-todo
   cd react-todo
   npm create vite@latest . -- --template react
   npm install
   ```

2. **Ask AI:**
   ```
   Create a React todo app with:

   - Mark as complete
   - Filter (all/active/completed)
   - Local storage persistence
   ```

3. **Replace App.jsx** with AI code

4. **Run:**
